# real-estate-price-prediction

This project analyzes housing sales data from King County, USA to understand the factors that influence house prices and build regression models for price prediction.

The analysis was completed using Python as part of the **IBM Data Analysis with Python course on Coursera (Skills Network Labs)**.

## Dataset

The dataset contains information about house sales in King County, including features such as price, bedrooms, bathrooms, square footage, floors, waterfront view, condition, and grade.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn
* Scikit-learn
* Jupyter Notebook

## What the Notebook Covers

The notebook includes loading and cleaning the dataset, performing exploratory data analysis to understand the relationships between different housing features and house prices, and building regression models such as linear and polynomial regression to predict prices. Model performance is evaluated using the R² score.
